## Getting Started

### Prerequisites
- Python 3.9+
- Node.js 16+
- Azure Speech Service subscription

### Installation

1. Clone the repository
```bash
git clone https://github.com/imohweb/AI-Powered-certifications-exams-practice-questions.git
cd AI-Powered-certifications-exams-practice-questions
```

2. Set up backend
```bash
cd backend
pip install -r requirements.txt
```

3. Set up frontend
```bash
cd frontend
npm install
```

4. Configure environment variables
```bash
# Create .env file in backend directory
AZURE_SPEECH_KEY=your_speech_service_key
AZURE_SPEECH_REGION=your_speech_service_region
AZURE_TRANSLATOR_KEY=your_translator_service_key
AZURE_TRANSLATOR_REGION=your_translator_service_region
AZURE_OPENAI_API_KEY=your_openai_service_key
AZURE_OPENAI_ENDPOINT=your_openai_service_endpoint
```

### Running the Application

1. Start the backend
```bash
cd backend
uvicorn app.main:app --reload
```

2. Start the frontend
```bash
cd frontend
npm start
```

## üîÄ Microsoft-Style Question Randomization

This application replicates the **exact same behavior** as Microsoft's official practice tests, where each retake presents different questions:

### **How It Works (Just Like Microsoft)**
- ‚úÖ **Question Pool**: 100+ questions generated per certification (larger pool)
- ‚úÖ **Session Selection**: 50 random questions selected per practice session
- ‚úÖ **Answer Shuffling**: Answer options randomized for each session
- ‚úÖ **Fresh Retakes**: Different questions appear when you retake the same certification
- ‚úÖ **Difficulty Balance**: Maintains proper distribution (30% easy, 50% medium, 20% hard)
- ‚úÖ **Smart Rotation**: Avoids immediate repeats while ensuring variety

### **Benefits**
- üéØ **Realistic Preparation**: Matches the actual Microsoft practice test experience
- üß† **Better Learning**: Prevents memorization, encourages real understanding
- üîÑ **Retake Value**: Each practice session provides fresh questions and perspectives
- üìä **Comprehensive Coverage**: All exam domains covered across multiple sessions

> **üéñÔ∏è Microsoft Certification Alignment**: This randomization system mirrors Microsoft's official practice assessment behavior, giving you the most authentic preparation experience possible.
